Before you begin, ensure you have the following:

  • Python 3.8+
  • Docker
  • API keys for:
    • Pinecone
    • SerpAPI (or Tavily/Bing)
    • Snowflake
  • claat installed (for viewing this Codelab)

Navigate to the data_processing/ folder and run:

python preprocess_electricity_report.py

This script will:

  • Load PDFs (2022‚Äì2025)
  • Chunk the data
  • Add Year/Quarter metadata

Still in data_processing/, embed the processed chunks:

python vectorize_data_with_metadata.py

This script will:

  • Embed the content using OpenAI embeddings
  • Push it to Pinecone with Year/Quarter metadata

Navigate to the LangGraph orchestrator backend:

cd mini_manus_ai
uvicorn app.main:app --reload

The FastAPI backend handles:

  • Routing queries to appropriate agents
  • Aggregating agent responses

In a new terminal:

cd frontend
streamlit run ui.py

From here, users can:

  • Ask research questions
  • Filter by Year or Quarter
  • Choose agents to run (RAG, Web, Image)

To containerize the full app:

docker build -t electricity-assistant .
docker run -p 8501:8501 electricity-assistant

Access the app at http://localhost:8501

Use the provided script to auto-generate reports:

python generate_testreport.py

This combines all agent outputs into a report format (text + visual).

You've built a full-stack research assistant that:

  • Embeds and retrieves unstructured report data
  • Uses LangGraph agents for coordination
  • Delivers insights and visual content
  • Runs locally or in Docker
  • Add confidence scoring
  • Use streaming responses from LLMs
  • Deploy to cloud using GitHub Actions or Streamlit Sharing
