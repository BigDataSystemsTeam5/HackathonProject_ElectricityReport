• Preprocess and embed electricity reports from 2022–2025
  • Use a LangGraph-based orchestrator with RAG, Web Search, and Image agents
  • Interact via a Streamlit + FastAPI app
  • Deploy using Docker

Prerequisites

  • Python 3.8+
  • Docker
  • Snowflake, Pinecone, and SerpAPI API keys

Step 1: Preprocess Electricity Reports

Navigate to the data_processing/ folder:

cd data_processing

Run the preprocessing script:

python preprocess_electricity_report.py

This will:

  • Load PDFs from multiple years
  • Chunk content and extract Year/Quarter metadata
  • Store formatted data ready for vector embedding

Step 2: Index Data to Pinecone

Inside the same folder, use the embedding script:

python vectorize_data_with_metadata.py

This pushes your processed data into Pinecone with metadata like:

  • year
  • quarter

You can verify data in your Pinecone project dashboard.

Step 3: Start the FastAPI Backend (LangGraph Agents)

Go to the backend code directory:

cd mini_manus_ai

Run the LangGraph FastAPI backend:

uvicorn app.main:app --reload

This starts:

  • RAG Agent (using Pinecone)
  • Web Agent (using SerpAPI)
  • Image Agent (DALL·E-3 via OpenAI)
  • AgentRouter using LangGraph orchestration

Step 4: Launch Streamlit UI

Open a new terminal and navigate to:

cd frontend
streamlit run ui.py

The UI allows:

  • Filtering results by year and quarter
  • Asking free-form research questions
  • Triggering any or all of the three agents

Step 5: Run the Full Stack with Docker

At the root directory, build and run the Docker container:

docker build -t electricity-assistant .
docker run -p 8501:8501 electricity-assistant

Step 6: Additional Utilities

To generate a test research report:

python generate_testreport.py

This outputs a consolidated markdown file combining all agent results for a given question.

Summary

✅ You've executed a full-stack, AI-powered research assistant that:

  • Parses and indexes electricity reports
  • Responds with summarized insights, visuals, and news
  • Uses LangGraph agents orchestrated in FastAPI
  • Is interactive via Streamlit and portable via Docker
