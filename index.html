You'll build an intelligent assistant that answers electricity-related research queries using a multi-agent system (RAG, Web Search, Snowflake, and Image generation). The app uses:

  • LangGraph to coordinate agents
  • Pinecone for vector search on Electricity PDFs
  • Snowflake for structured analytics
  • SerpAPI for real-time news
  • OpenAI (GPT-4o + DALL¬∑E) for reasoning and visuals
  • FastAPI + Streamlit for API & UI
  • Docker for deployment
 
git clone https://github.com/BigDataSystemsTeam5/HackathonProject_ElectricityReport.git
cd HackathonProject_ElectricityReport

Create a .env file in the root directory:

# Pinecone
PINECONE_API_KEY=your_pinecone_api_key  
PINECONE_ENV=your_pinecone_env  
PINECONE_INDEX=your_index  

# Snowflake
SNOWFLAKE_USER=your_user  
SNOWFLAKE_PASSWORD=your_password  
SNOWFLAKE_ACCOUNT=your_account  
SNOWFLAKE_DATABASE=your_db  
SNOWFLAKE_SCHEMA=your_schema  
SNOWFLAKE_WAREHOUSE=your_warehouse  

# OpenAI + Web Search
OPENAI_API_KEY=your_openai_key  
SERPAPI_API_KEY=your_serpapi_key

(Optional) Create a virtual environment:

python -m venv venv  
source venv/bin/activate   # Windows: venv\Scripts\activate

Install required packages:

pip install -r requirements.txt

Navigate to the data_processing/ folder and run:

python load_reports_to_pinecone.py
  • rag_agent.py: Retrieves relevant PDF chunks from Pinecone
  • web_search_agent.py: Fetches live data via SerpAPI
  • snowflake_agent.py: Pulls structured metrics from Snowflake
  • image_agent.py: Creates illustrations using GPT-4o + DALL¬∑E
  • multi_agent_graph.py: Coordinates all agents with LangGraph
 
uvicorn backend.api:app --reload

Example POST:

{
  "question": "What happened in Q1 2023?",
  "year": 2023,
  "quarter": "Q1"
}

cd frontend
streamlit run streamlit_app.py

docker build -t research-assistant .
docker run -p 8501:8501 research-assistant
